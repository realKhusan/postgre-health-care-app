Welcome to the Health Care Management System

This application allows you to manage Patients and their Appointments using a RESTful API structure.

📋 Patient Module

  • GET → /api/patients/all – Retrieve all patients
  • POST → /api/patients/add – Add a new patient
  • PUT → /api/patients/edit/:id – Edit a patient by ID
  • DELETE → /api/patients/delete/:id – Delete a patient by ID
  • GET → /api/patients/without-appointments – Get patients without appointments

📅 Appointment Module

  • GET → /api/appointments/all – Retrieve all appointments
  • POST → /api/appointments/add – Add a new appointment
  • PUT → /api/appointments/edit/:id – Edit an appointment by ID
  • DELETE → /api/appointments/delete/:id – Delete an appointment by ID
  • GET → /api/appointments/by-date – Filter appointments by date range
  • GET → /api/appointments/by-doctor/:doctor – Appointments by doctor

🛡️ Data Validation

All input data is validated to ensure:

  • ✔ Required fields are filled
  • ✔ Dates and emails have valid formats
  • ✔ Unique constraints and logic rules are respected

Use tools like Postman to send requests to these routes.
